## Boston Airbnb App

This app displays Airbnb data of Boston.

### Technologies Used
Frontend:
- React App

Backend:
 - Node/Express

Data:
 - https://www.kaggle.com/airbnb/boston

Database:
- MySQL

Containers:
 - Docker
